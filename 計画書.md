# 翻訳補助アプリ 計画書

## 1. 目的

Zoomで英語圏の人と会議をする際の、言語の壁を下げるための補助ツール。
有料の翻訳ツールに頼らず、ブラウザ標準機能＋無料APIで実現する。

## 2. 全体ゴール（最終形）

| 機能 | 内容 | 開発フェーズ |
|------|------|------------|
| スピーキング補助 | 自分の日本語 → 英語に翻訳・表示・読み上げ | **フェーズ1（今回）** |
| リスニング補助 | 相手の英語 → 日本語字幕で表示 | フェーズ2（後日） |

今回の計画書は **フェーズ1（スピーキング補助）** を対象とする。

## 3. フェーズ1のスコープ

### やること
- マイクから日本語を音声認識（聞き取り）
- 認識した日本語を英語へ自動翻訳
- 翻訳結果を画面に大きく表示
- 翻訳結果を英語音声で読み上げ
- 認識テキストの手動編集に対応
- 直近のやり取りを履歴として保持

### やらないこと（フェーズ1では対象外）
- 相手の英語音声の取り込み（フェーズ2）
- 翻訳結果のZoomへの自動入力
- 会話全体の議事録保存・エクスポート
- ログイン・アカウント機能

## 4. 技術構成（すべて無料・APIキー不要）

| 役割 | 使用技術 | 料金 | 備考 |
|------|---------|------|------|
| 音声認識（日→文字） | Web Speech API (`webkitSpeechRecognition`) | 無料 | Chrome必須 |
| 翻訳（日→英） | MyMemory翻訳API | 無料 | 匿名で1日上限あり／メール登録で上限増 |
| 音声合成（英読み上げ） | Web Speech API (`SpeechSynthesis`) | 無料 | キー不要 |
| 実行環境 | 単一HTMLファイル（Chromeで開くだけ） | 無料 | サーバー不要 |

> 注意: Claude APIは使わない。Claude Codeのサブスクと、アプリから叩くClaude APIの課金は別物のため、「無料」を守るためブラウザ標準機能と無料APIで構成する。

## 5. 処理フロー

```
[マイク] 日本語で話す
   ↓ Web Speech API
日本語テキスト（画面表示・編集可）
   ↓ MyMemory API
英語テキスト（画面に大きく表示）
   ↓ SpeechSynthesis
英語の読み上げ音声
   ↓
履歴に追加
```

## 6. 画面構成

1. マイク操作エリア（話し始める／停止、状態表示）
2. 日本語エリア（聞き取り結果・編集可能なテキスト欄）
3. 英語エリア（翻訳結果を大きく表示、再読み上げボタン）
4. 操作設定（自動翻訳ON/OFF、自動読み上げON/OFF、読み上げ速度）
5. 履歴エリア（過去のやり取り、各行に再読み上げボタン）
6. 設定（任意のメール登録、使い方説明）

## 8. 制約・既知のリスク

- **Chrome必須**: Web Speech APIはChrome系ブラウザのみ安定動作。
- **翻訳精度**: MyMemoryは無料ゆえ、DeepL等より精度は劣る。重要な場面では英文を目視確認する。
- **翻訳の利用上限**: 匿名利用は1日あたり制限あり。メール登録で上限を拡大できる。
- **日本語認識精度**: 騒音・早口で精度が落ちる。短文で区切って話すと改善。
- **マイク許可**: 初回にブラウザのマイク使用許可が必要。

## 9. Zoomでの使い方（想定運用）

- PCスピーカーから出る英語の読み上げ音声を相手に届ける、または
- 画面の英文を相手に見せて伝える（画面共有 or 読み上げ）
