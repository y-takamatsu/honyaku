# VoiceMeeter Banana セットアップ手順書

リスニング補助アプリ（`listening-helper.html`）でZoomの相手の英語を字幕表示するために、PCに **VoiceMeeter Banana** という仮想オーディオツールをインストールします。

> ⚠️ **作業時間の目安**: 約20分（ダウンロード5分 + インストール5分 + 設定10分）

---

## 0. このツールについて（事前知識）

| 項目 | 内容 |
|------|------|
| ツール名 | VoiceMeeter Banana |
| 開発元 | VB-Audio Software（フランス） |
| 種類 | 仮想オーディオミキサー |
| 対応OS | Windows XP〜11（32/64bit・ARM64） |
| 料金 | **無料**（寄付歓迎・全機能使用可。30日後に寄付の案内が出るが使い続けてOK） |
| データ収集 | なし／インターネット接続不要 |
| なぜ必要か | Zoomから出る音をブラウザに流し込むため |

---

## 1. ダウンロード

### 方法A: 公式サイトから（推奨）

1. ブラウザで以下のURLを開く
   👉 https://vb-audio.com/Voicemeeter/banana.htm
2. ページ中段にある「**VoicemeeterSetup_v2122.zip**」のダウンロードボタンをクリック
3. ZIPファイル（約42MB）がダウンロードされる

### 方法B: 直接ダウンロード

以下のリンクをクリックすると、ZIPファイルを直接ダウンロードできます:
👉 https://download.vb-audio.com/Download_CABLE/VoicemeeterSetup_v2122.zip

> ※ファイル名のバージョン番号（v2122）は更新で変わることがあります。公式サイトに最新版があります。

---

## 2. インストール

### STEP 2-1: ZIPを解凍

1. ダウンロードフォルダで `VoicemeeterSetup_v2122.zip` を **右クリック → すべて展開**
2. 展開先（どこでもOK）を選んで「展開」をクリック
3. 解凍されたフォルダの中に `VoicemeeterSetup.exe` がある

### STEP 2-2: 管理者として実行

1. `VoicemeeterSetup.exe` を **右クリック → 「管理者として実行」**

   > 🚨 重要: 普通のダブルクリックではなく **「管理者として実行」** を使ってください。仮想オーディオドライバを入れるために管理者権限が必要です。

2. 「このアプリがデバイスに変更を加えることを許可しますか？」と聞かれたら「**はい**」
3. インストーラ画面で「**Install**」をクリック
4. インストール完了画面が出たら閉じる

### STEP 2-3: PCを再起動 ⚠️ 必須

仮想オーディオドライバを有効にするため、**必ずPCを再起動してください**。

---

## 3. VoiceMeeter Banana の初期設定

### STEP 3-1: VoiceMeeter Banana を起動

1. スタートメニューで「**VoiceMeeter Banana**」を検索して起動
   - ※「VoiceMeeter」（バナナなし）も一緒にインストールされていますが、こちらは使いません。間違えないよう注意。
2. 初回起動時にチュートリアル画面が出たら閉じる

### STEP 3-2: 画面の見方（ざっくり）

VoiceMeeter Banana の画面は3つのエリアに分かれています:

```
┌───────────────────┬──────────────────┬─────────────────┐
│  HARDWARE INPUT   │  VIRTUAL INPUT   │  HARDWARE OUT   │
│  (実マイク等)      │  (仮想入力)       │  A1 A2 A3       │
│  ※今回使わない    │  ★ここに         │  ★ここに       │
│                   │  Zoom音声が来る   │  ヘッドホン設定 │
└───────────────────┴──────────────────┴─────────────────┘
```

### STEP 3-3: 出力先（A1）にヘッドホン/スピーカーを設定

1. 画面の **右上にある「A1」** と書かれたボタン（▼マーク付き）をクリック
2. メニューが出るので、**自分のヘッドホン/スピーカー** を選ぶ
   - 例: 「Realtek High Definition Audio」「USB Headphones」など
   - 名前の先頭に `WDM:` がついているものを選ぶと安定

### STEP 3-4: 仮想入力の出力ボタンを点灯

1. 中央エリアにある **「VoiceMeeter Input」** という縦長のストリップを見る（"VIRTUAL INPUTS" の左側）
2. このストリップの下部に **A1, A2, A3, B1, B2** というボタンが並んでいる
3. **「A1」と「B1」の2つだけ点灯**（緑色）させる
   - A1 = あなたのヘッドホンに音を流す
   - B1 = アプリ（Chrome）に音を流す
4. 他のボタン（A2, A3, B2）は消灯のまま

---

## 4. Windows の音声設定

### STEP 4-1: 既定の出力デバイスを変更

1. タスクバー右下の **🔊スピーカーアイコンを右クリック** → 「サウンドの設定」
2. 「**出力**」セクションを探す
3. ドロップダウンから **「VoiceMeeter Input (VB-Audio VoiceMeeter VAIO)」** を選ぶ

   > これで、Windows全体の音（Zoom含む）が VoiceMeeter Banana に流れ込みます。

### STEP 4-2: 音が聞こえるか確認

1. YouTubeなどで何か音を再生
2. 自分のヘッドホン/スピーカーから音が聞こえればOK

   > 聞こえない場合は、VoiceMeeter Banana の画面で **A1 のフェーダー** が下まで下がっていないか確認（フェーダーは中央〜上が標準）。

---

## 5. Chrome 側の設定

### STEP 5-1: リスニング補助アプリを開く

1. `listening-helper.html` を Chrome で開く（ダブルクリック）
2. 「🎧 聞き取り開始」ボタンをクリック

### STEP 5-2: マイク選択ダイアログでVoiceMeeter Outputを選ぶ

1. Chrome がマイクの使用許可を求めてきたら、**マイク選択ドロップダウン** をクリック
2. **「VoiceMeeter Output (VB-Audio VoiceMeeter VAIO)」** を選ぶ

   > ⚠️ 「VoiceMeeter **Input**」ではなく「VoiceMeeter **Output**」が正解。逆だと動きません。

3. 「**許可**」をクリック

### STEP 5-3: マイク選択を覚えさせる（任意）

毎回選び直すのが面倒なら:

1. Chromeのアドレスバー左の **🔒鍵マーク** をクリック
2. 「サイトの設定」
3. 「マイク」セクションで **VoiceMeeter Output** を選択
4. ページを再読み込み

これで次回からは自動的にVoiceMeeter Outputが使われます。

---

## 6. 動作確認

### テスト手順

1. YouTubeで適当な英語動画を再生（例: TED Talks）
2. リスニング補助アプリ画面で:
   - 上部 **「英語（聞き取り中の音声）」** に英文が流れる
   - 中央 **「日本語字幕」** に翻訳が大きく表示される
3. うまく動けば本番のZoom会議でも同じ仕組みで動きます

## 7. トラブルシューティング

### ❌ Zoom の音が自分に聞こえない

- VoiceMeeter Banana 画面で「VoiceMeeter Input」ストリップの **A1 が点灯（緑）** か確認
- 「A1」フェーダーが下まで下がっていないか
- 右上の「A1」に設定したデバイスが正しいか

### ❌ アプリに英語が流れてこない

- VoiceMeeter Banana 画面で「VoiceMeeter Input」ストリップの **B1 が点灯（緑）** か確認
- Chrome のマイク設定が **VoiceMeeter Output** になっているか（鍵マークから確認）
- Windowsの既定の出力デバイスが **VoiceMeeter Input** になっているか
- YouTube等で音を出してみて、本当に音声がVoiceMeeterに流れているか

### ❌ Chrome のマイク選択にVoiceMeeterが出てこない

- PCを **再起動** したか確認（仮想ドライバは再起動で有効になる）
- Chromeを完全終了 → 起動し直し
- それでも出ない場合は、VoiceMeeterのインストールが失敗している可能性 → アンインストールして再インストール

### ❌ 音が二重に聞こえる / エコーがかかる

- Windows のサウンド設定で、既定の出力が **VoiceMeeter Input** 「だけ」になっているか確認
- ヘッドホンやスピーカーが他のアプリで直接使われていないか

### ❌ 30日後に寄付の案内画面が出る

- 「Donate Later」「Cancel」で閉じれば、機能制限なく使い続けられます
- 気に入ったら寄付するのが礼儀ですが、強制ではありません

---

## 8. 普段使い（2回目以降）

セットアップが終われば、次回からは以下のように簡単です:

1. PC起動時に VoiceMeeter Banana が自動起動する（タスクトレイで動作中）
2. Zoom を起動する → そのまま音はVoiceMeeter経由で流れる
3. `listening-helper.html` を Chrome で開く
4. 「聞き取り開始」をクリック → 字幕表示開始

> 💡 VoiceMeeter Banana を自動起動させるには、設定メニュー（右上歯車）→「System Tray (On Startup)」を有効に。

---

## 9. 元の音声設定に戻したいとき

リスニング補助を使わないときに、通常のヘッドホン直接出力に戻したい場合:

1. タスクバー右下の **🔊スピーカーアイコン** をクリック
2. **出力デバイス** を VoiceMeeter Input から **自分のヘッドホン** に戻す

> アンインストールしたい場合は、コントロールパネル → プログラムと機能 → VoiceMeeter → アンインストール → **PC再起動**。
